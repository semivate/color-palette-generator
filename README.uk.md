# 🎨 Генератор палітри кольорів

### [Live Demo](https://semivate.github.io/color-palette-generator/)

Простий генератор кольорових палітр створений на JavaScript.  
Дозволяє генерувати випадкові кольори та швидко копіювати HEX-коди в буфер обміну.

---

## Функції

- Генерація випадкової палітри з 5 кольорів.
- Копіювання HEX-коду одним кліком.
- Візуальне підтвердження копіювання.
- Анімована кнопка генерації.
- Зміна фону сторінки при виборі кольору.
- Мінімалістичний UI.

---

## Використані технології

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
